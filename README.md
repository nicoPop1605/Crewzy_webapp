# 🎉 Crewzy

> A collaborative social event management application designed to simplify calendar planning, group creation, and friend management.

## ✨ Features

* **Calendar Planning:** Intuitively schedule, view, create, and manage upcoming events.
* **Group Creation:** Organize different social circles, coordinate gatherings, and communicate effortlessly within dedicated groups.
* **Friend Management:** Connect with friends, keep track of connections, and streamline the invitation process.
* **Free Days Indicator:** Mark your free days to let friends know you are available and see who else would like to make plans.
* **Event Discovery:** Discover events taking place in the city (e.g., karaoke night, beer pong, etc.).
* **Real-time Statistics:** Monitor your event engagement with real-time percentages for accepted, pending, and declined invitations.

## 🛠️ Tech Stack

* **Frontend:** React, TypeScript
* **Backend:** Node.js, GraphQL, Prisma
* **Testing & Security:** Faker (utilized to simulate data and test application resilience against bot attacks)
